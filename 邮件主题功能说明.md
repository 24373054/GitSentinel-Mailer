# 🎨 邮件主题功能说明

## ✅ 功能已完成

您现在可以为每个监控项目选择不同的邮件通知风格！

## 🎯 可用主题

### 1. 默认主题（蓝紫渐变）
**原有风格**
- 🎨 蓝紫渐变背景
- 💫 现代化AI风格
- 适合：通用项目

### 2. 瀛州纪（赛博史诗）⭐ 新增
**完全匹配您的游戏风格**

- ✨ 青蓝霓虹发光效果 (#00d4ff)
- 📜 矩阵网格背景
- 🔮 账本式设计
- 💎 等宽字体显示代码
- 主题词：链上记录、数字碑文、永恒见证

### 3. 简约黑白
**极简主义设计**
- ⚫ 纯黑白配色
- 📄 干净利落
- 适合：专业项目

### 4. 温暖橙色
**温暖友好风格**
- 🟠 橙色渐变
- ☀️ 阳光感觉
- 适合：社区项目

### 5. 清新绿色
**自然清新风格**
- 🟢 绿色系
- 🌱 生机勃勃
- 适合：环保、健康相关项目

## 🚀 使用方法

### 添加新项目时选择主题

1. 填写仓库地址和邮箱
2. 在"邮件主题"下拉框中选择风格
3. 点击"添加项目"

### 修改现有项目主题

1. 停止项目监控
2. 点击"编辑"按钮
3. 更改"邮件主题"选项
4. 点击"保存修改"
5. 重新启动监控

## 🎨 瀛州纪主题详细说明

### 视觉元素

1. **色彩**
   - 背景：深邃蓝黑 (#0a0e27)
   - 主色：青蓝霓虹 (#00d4ff)
   - 强调：金色 (#ffd700) 用于图标
   - 紫色 (#6b46c1) 用于提示框

2. **特效**
   - 文字发光：`text-shadow: 0 0 20px cyan`
   - 边框发光：`box-shadow: 0 0 20px cyan`
   - 矩阵网格背景

3. **排版**
   - 标题使用等宽字体
   - 代码式SHA显示
   - 左侧账本线条装饰

4. **文案**
   - "链上记录"替代"变更通知"
   - "数字碑文"强调永恒性
   - "账本"隐喻区块链

### 适用场景

✅ **推荐使用于：**
- 《瀛州纪》游戏相关仓库
- 区块链项目
- 科技感强的项目
- 需要营造史诗感的项目

❌ **不太适合：**
- 轻松娱乐类项目
- 传统企业项目
- 需要亲和力的社区项目

## 💾 数据库存储

主题选择会保存在数据库中：
- 字段：`email_theme`
- 默认值：`'default'`
- 可选值：`default`, `yingzhouji`, `minimal`, `warm`, `fresh`

## 🔄 修改主题的影响

### 立即生效
- ✅ 下次发送邮件时使用新主题
- ✅ 无需重启服务

### 不影响
- ✅ 已发送的邮件保持原样
- ✅ 监控状态不变
- ✅ 提交记录不变

## 🛠️ 技术实现

### 后端
- `server/emailThemes.js` - 主题配置和HTML生成器
- `server/emailService.js` - 邮件发送服务
- `server/database.js` - 存储主题选择

### 前端
- `ProjectForm.js` - 主题选择下拉框
- `ProjectList.js` - 显示当前主题
- API调用获取主题列表

### 主题结构
```javascript
{
  name: '主题名称',
  colors: { /* 颜色配置 */ },
  gradient: '渐变样式',
  generateHTML: function() { /* HTML生成逻辑 */ }
}
```

## 📝 添加自定义主题

如需添加新主题，编辑 `server/emailThemes.js`：

```javascript
// 添加新主题
newtheme: {
  name: '新主题名称',
  colors: {
    primary: '#颜色值',
    secondary: '#颜色值',
    // ...更多颜色
  },
  gradient: 'linear-gradient(...)',
  generateHTML: function(repoUrl, commits, theme) {
    return `<!-- 自定义HTML模板 -->`;
  }
}
```

## ⚙️ 配置选项

### 瀛州纪主题可调整项

在 `server/emailThemes.js` 中可调整：
- 发光强度：`text-shadow`参数
- 网格密度：`repeating-linear-gradient`间距
- 边框颜色：`border`和`box-shadow`
- 字体大小：各元素的`font-size`
